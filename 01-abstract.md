
# Abstract
This project presents a complete IoT-based environmental monitoring system using an ESP8266 NodeMCU microcontroller integrated with a DHT11/DHT22 sensor.

The system continuously reads environmental parameters at 10-second intervals and transmits real-time data to the ThingSpeak cloud platform.

## Features
- Real-time monitoring
- Cloud visualization
- ON/OFF control system
- Low cost implementation

## Performance
- 99.2% transmission success
- 99.8% uptime

## Applications
Smart homes, greenhouses, laboratories.
